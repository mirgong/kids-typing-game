# 小键盘探险家 - 儿童打字游戏网站

## 🎮 项目介绍
一个专为5-10岁儿童设计的卡通风格打字学习网站，帮助小朋友学习键盘和鼠标使用。

## 🌐 在线访问
Gitee Pages地址：`https://你的用户名.gitee.io/kids-typing-game/`

## 🚀 快速开始
1. 直接访问上方链接即可使用
2. 无需登录，无需安装
3. 适合5-10岁儿童

## 📱 功能特点
- **卡通风格界面**：适合儿童的鲜艳色彩和动画
- **循序渐进学习**：字母 → 单词 → 句子
- **趣味游戏**：字母雨、单词赛车、打字冒险
- **实时反馈**：键盘高亮显示，音效提示
- **进度跟踪**：成就系统和学习统计

## 🛠️ 本地运行
```bash
# 方法1：直接打开
双击 index.html 文件

# 方法2：使用HTTP服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
