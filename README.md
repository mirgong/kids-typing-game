# 小键盘探险家 - 儿童打字游戏网站

一个专为5-10岁儿童设计的卡通风格打字学习网站，帮助小朋友学习键盘和鼠标使用。

## 功能特点

### 🎯 核心功能
- **循序渐进的学习路径**：从字母到单词再到句子
- **趣味游戏化学习**：通过游戏激发学习兴趣
- **实时反馈系统**：即时提示和鼓励
- **进度跟踪**：记录学习成果和成就

### 🎨 设计特色
- **卡通风格界面**：适合儿童的色彩和动画
- **动物角色陪伴**：猴子、熊猫、兔子等可爱角色
- **交互式键盘显示**：实时高亮显示按键
- **音效反馈**：正确/错误/完成都有相应音效

### 🚀 技术特点
- **纯前端实现**：无需后端服务器
- **响应式设计**：适配各种设备
- **本地存储**：学习进度保存在浏览器中
- **无登录要求**：直接开始学习

## 课程体系

### 1. 字母探险（初级）
- 学习基本字母键位：A, S, D, F, J, K, L, ;
- 每个字母都有详细提示
- 实时键盘高亮显示

### 2. 单词森林（中级）
- 练习简单单词：CAT, DOG, SUN, MOON等
- 培养单词拼写能力
- 提高打字速度

### 3. 句子城堡（高级）
- 挑战完整句子打字
- 学习基本语法结构
- 提高综合打字能力

## 游戏模式

### 🎮 字母雨
- 字母从屏幕上方掉落
- 输入正确字母接住它们
- 锻炼反应速度和准确性

### 🏎️ 单词赛车
- 输入单词让赛车前进
- 与时间赛跑
- 提高单词输入速度

### 🐉 打字冒险
- 通过打字打败怪物
- 拯救小动物
- 故事化学习体验

## 部署说明

### 简单部署（推荐）
1. 将整个文件夹上传到任何静态网站托管服务
2. 无需任何配置，直接访问

### 支持的托管服务
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- 任何支持静态网站的服务器

### 本地运行
1. 下载所有文件到同一目录
2. 使用任何HTTP服务器运行
   ```bash
   # 使用Python
   python3 -m http.server 8000
   
   # 使用Node.js
   npx serve .
   
   # 使用PHP
   php -S localhost:8000
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 自定义配置

### 修改课程内容
编辑 `script.js` 文件中的 `gameData.lessons` 对象：
```javascript
lessons: {
    1: {
        title: '你的课程标题',
        challenges: [
            { type: 'letter', target: 'A', hint: '你的提示文字' },
            // 更多挑战...
        ]
    }
}
```

### 修改颜色主题
编辑 `style.css` 文件中的颜色变量：
```css
/* 主要颜色 */
--primary-color: #ff6b6b;
--secondary-color: #4ecdc4;
--accent-color: #ffd166;
```

### 添加新游戏
在 `script.js` 的 `gameData.games` 对象中添加新游戏：
```javascript
games: {
    'your-game-id': {
        title: '游戏标题',
        description: '游戏描述'
    }
}
```
